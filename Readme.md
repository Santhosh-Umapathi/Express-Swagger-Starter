## Installation

Express Auth System App requires [Node.js](https://nodejs.org/) or [Yarn](https://yarnpkg.com/) to run.

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

- Swagger Docs
- YAML
- File Upload, Multiple File Uploads
- Cloudinary
